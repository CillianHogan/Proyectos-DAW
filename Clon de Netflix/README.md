Clon de Netflix - Maquetación y Lógica JS
Este proyecto nació como un ejercicio para calcar la interfaz de Netflix y practicar cómo hacer una web que sea totalmente responsive.

Lo que he trabajado aquí:
Maquetación fiel: Seguí un tutorial para la estructura de HTML y el CSS. Quería que quedara visualmente igual, así que me centré en los detalles de las sombras y el posicionamiento de los elementos.

El carrusel (Swiper.js): Fue mi primera vez usando una librería externa para sliders. Me costó bastante configurar los breakpoints para que el número de películas cambiara bien entre móvil y PC sin que se rompiera el diseño.

Mis añadidos en JS: El tutorial era muy visual, así que yo le metí la lógica por mi cuenta. Programé la validación del email con expresiones regulares y añadí el sonido "Tudum" al darle al botón de registro para que la experiencia fuera más realista.

Eventos de usuario: Configuré el código para que al pulsar la tecla Enter el formulario se envíe solo, algo que parece una tontería pero que mejora mucho el uso de la web.

Tecnologías usadas:
HTML5 y CSS3 (Flexbox y Grid).

JavaScript Vanilla para toda la interactividad.

Swiper.js para los sliders de las carátulas.

Audio API para los efectos de sonido.

Créditos:
La estructura visual y los estilos están basados en el tutorial de https://www.youtube.com/@codigoam.
