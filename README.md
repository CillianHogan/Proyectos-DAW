Aquí he ido subiendo los proyectos que he hecho para el grado de DAW. En cada uno me he centrado en aprender una cosa distinta:

La PokéDex: Aquí el lío fue conectar con la PokéAPI y que no fuera lenta. Le añadí un buscador para encontrar los Pokémon por nombre al momento y me busqué la vida para que también se pudieran escuchar sus gritos originales.

Lista de Tareas (To-Do): Lo más importante es que las tareas se guardan en el LocalStorage (para que no se borren al refrescar) y que puedes filtrarlas por prioridad para organizarte mejor.

Tienda ElectroShop: Un ejercicio para practicar cómo pintar una tienda usando datos de una API externa. Tiene carrito de compra y está hecha con CSS Grid para que se vea bien en cualquier pantalla.

Clon de Netflix: Un proyecto de maquetación para que quedara lo más parecido posible a la plataforma real. Usé la librería Swiper.js para los sliders de las pelis y le metí algo de lógica en el registro con sonidos y validaciones.


Ver proyectos en vivo: https://cillianhogan.github.io/Proyectos-DAW/
