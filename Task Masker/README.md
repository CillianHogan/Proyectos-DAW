Task-Masker - Proyecto de Gestión de Tareas
Este es un proyecto personal que he desarrollado para poner en práctica la lógica de JavaScript sin depender de un tutorial paso a paso. La idea era crear una lista de tareas sencilla pero que funcionara bien de verdad.

Lo que he hecho en este proyecto:
Manejo de la lista: He trabajado con un array de objetos para guardar cada tarea con su nombre y su prioridad. Me sirvió para entender cómo añadir y borrar elementos de una lista de forma dinámica.

Guardado de datos (LocalStorage): Para que no se borraran las tareas al refrescar la página, usé el almacenamiento del navegador. He aprendido a pasar los datos a texto (JSON) para que el navegador los pueda guardar y recuperar.

Filtros básicos: He creado unos botones para filtrar las tareas. No ha sido fácil cuadrar que se muestren solo las "Altas" o las "Hechas" sin que el resto desaparecieran del array, pero al final lo saqué con unas condiciones en el código.

Diseño con Tailwind: He usado Tailwind CSS para no complicarme con archivos de estilo gigantes y poder darle un aspecto limpio y moderno directamente desde el HTML.

Tecnologías:
JavaScript: Lógica de arrays y manipulación del DOM.

LocalStorage: Para la persistencia de los datos.

Tailwind CSS: Para el diseño visual.
